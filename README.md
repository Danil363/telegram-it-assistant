# Telegram IT Assistant (RAG-based)

Telegram IT Assistant is an intelligent Telegram bot built using a
Retrieval-Augmented Generation (RAG) architecture.

The project combines a large language model with a vector-based knowledge
retrieval system to provide accurate, context-aware, and grounded answers
to IT-related questions.

---

Features

- Telegram bot interface
- Retrieval-Augmented Generation (RAG)
- Semantic search using embeddings
- Knowledge base built from documents and text data
- Context-aware answers based on retrieved sources
- Modular and extensible architecture

---



Tech Stack

- **Python**
- **Telegram Bot API**
- **Large Language Model (LLM)**
- **Embeddings**
- **Vector Database** (FAISS)
- **Async / FastAPI** (optional backend)
